# Diabetes Prediction using Machine Learning

This project predicts whether a person is diabetic using a machine learning model.

## Dataset
Pima Indians Diabetes Dataset

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Model
Logistic Regression

## Steps
1. Load dataset
2. Preprocess data
3. Train model
4. Evaluate accuracy
5. Predict diabetes risk
## Run the Web App

1. Install dependencies
pip install -r requirements.txt

2. Run the application
python -m streamlit run app.py
